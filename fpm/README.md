# FPM

## Exam Strategies

There seems to always be the following skills tested for analysis:
- Proving sequence convergence or boundedness
- Series convergence (often with taylor series)
- Proving continuity
- Prove a limit
- Basically just application of mean value theorem

There seems to always be the following tested used for algebra:
- Here's a set and operation, prove it's a group and verify these properties
- Here's a group, either prove this is a subgroup or find all subgroups
- Prove an isomorphism between groups
- An actions question that asks if the action is transitive and faithful
